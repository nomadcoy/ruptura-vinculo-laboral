# Ruptura do Vínculo Laboral

Este projeto realiza uma análise sociológica da ruptura do vínculo laboral
a partir de dados organizacionais de uma empresa de grande porte.

A hipótese central é que a saída do trabalhador não pode ser compreendida
como decisão individual isolada, mas como resultado de trajetórias
socialmente estruturadas por hierarquia, intensificação do trabalho
e desigualdades institucionais.

## Base de dados
- HR Employee Attrition (Kaggle)

## Ferramentas
- Python
- Pandas
- Plotly
- Jupyter Notebook

## Estrutura do projeto
- `notebook.ipynb` – análise principal
- `WA_Fn-UseC_-HR-Employee-Attrition.csv` – base de dados
- `README.md` – apresentação do projeto

## Conclusão
A ruptura do vínculo laboral emerge como um sintoma das condições sociais
de permanência que a organização é capaz — ou incapaz — de oferecer.
